# M005 — 快捷聊天与官网

## Context

产品在高频触达和外部品牌触点方面仍需加强。

## Outcome

- 提供元宝风格快捷呼出框用于快速聊天。
- 完成官网制作并对外承载产品信息与下载入口。

## Slices

### M005-S01 — 快捷呼出框

- [ ] T-M005-S01-001：定义呼出框交互规范（唤起、收起、焦点、会话同步）。
- [ ] T-M005-S01-002：实现系统级快捷键呼出与主窗口状态联动。
- [ ] T-M005-S01-003：优化小窗性能与输入体验。

验收标准：
- 可通过快捷方式稳定呼出与收起。
- 呼出框与主聊天会话内容一致。

### M005-S02 — 官网制作

- [ ] T-M005-S02-001：确定官网信息架构（价值、能力、下载、更新）。
- [ ] T-M005-S02-002：完成视觉与前端实现。
- [ ] T-M005-S02-003：接入版本与下载信息更新机制。

验收标准：
- 官网可完整表达产品价值与核心能力。
- 下载与版本信息可维护。

## Dependencies

- 依赖 M001 确定最终视觉与信息架构语言。

## Risks

- 官网与桌面端品牌不一致会造成认知割裂，需要统一设计 token。

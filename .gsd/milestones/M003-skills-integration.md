# M003 — 技能安装一体化

## Context

技能能力已具备基础，但缺少面向角色装配、安全审查与测试闭环的一体化体验。

## Outcome

- 技能安装集成覆盖角色安装、安全审查、记忆管理、OpenViking、Turix、操作电脑界面、网站 CLI/OpenCLI。
- 支持通过命令进行角色安装。
- 角色安装测试链路可验证、可回滚。

## Slices

### M003-S01 — 技能集成入口与分类

- [ ] T-M003-S01-001：建立技能安装中心，按角色/安全/记忆/工具分类展示。
- [ ] T-M003-S01-002：补齐技能来源、权限说明、启用开关。
- [ ] T-M003-S01-003：接入 OpenViking、Turix、OpenCLI 等目标能力。

验收标准：
- 目标技能可检索、可安装、可启停。
- 每个技能有清晰风险说明和状态反馈。

### M003-S02 — 角色安装命令化

- [ ] T-M003-S02-001：实现 /dragonclaw-agent-skill-install 命令入口。
- [ ] T-M003-S02-002：完成角色信息解析、参数校验、幂等处理。
- [ ] T-M003-S02-003：安装失败自动回退并输出诊断信息。

验收标准：
- 命令可基于角色信息完成技能装配。
- 重复执行不产生脏状态。

### M003-S03 — 安装测试与质量闸门

- [ ] T-M003-S03-001：覆盖技能安装成功/失败/重试/回滚测试。
- [ ] T-M003-S03-002：补齐角色安装场景测试矩阵。
- [ ] T-M003-S03-003：建立安装日志与审计输出标准。

验收标准：
- 关键安装路径有自动化验证。
- 失败路径具备可追踪证据。

## Dependencies

- 依赖 M002 的模型与登录基础已稳定。

## Risks

- 技能来源多样，安全边界和权限模型需严格定义。

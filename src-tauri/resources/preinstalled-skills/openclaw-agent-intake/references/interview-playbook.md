# Interview Playbook

用于字段缺失时的 Inversion 追问。

## Priority Order
1. 核心使命（角色为什么存在）
2. 必须/禁止规则（边界）
3. 交付物格式（输出可验收）
4. 流程步骤（执行顺序）
5. 风格与记忆策略（稳定性）

## Question Templates
- 使命缺失：
  - "这个角色最核心要优化的结果是什么？请给 1-3 条。"
- 边界缺失：
  - "这个角色必须遵守什么？绝对不能做什么？"
- 交付缺失：
  - "希望角色最终输出成什么结构（例如报告、计划、清单）？"
- 流程缺失：
  - "请给这个角色的最小执行步骤（至少 4 步）。"

## Constraints
- 一轮最多 3 个问题
- 不重复询问已确认字段
- 对高风险字段禁止默认猜测
